# Грамматика Thema 4 - Vervoer

---

## 1. Want и Omdat (Потому что)

Оба слова означают "потому что", но порядок слов **разный**!

### Want — обычный порядок слов

После **want** порядок слов остаётся **нормальным**: подлежащее + глагол.

```
[Главное предложение] + want + [Подлежащее + Глагол + Остальное]
```

| Главное предложение | want | Подлежащее | Глагол | Остальное |
|---------------------|------|------------|--------|-----------|
| Ik ga slapen | want | ik | **ben** | moe |
| Ik neem de bus | want | het | **regent** | hard |
| Ik bel niet | want | ik | **heb** | geen tijd |

### Omdat — глагол в конец!

После **omdat** глагол уходит **в конец** предложения.

```
[Главное предложение] + omdat + [Подлежащее + Остальное + Глагол]
```

| Главное предложение | omdat | Подлежащее | Остальное | Глагол |
|---------------------|-------|------------|-----------|--------|
| Ik ga slapen | omdat | ik | moe | **ben** |
| Ik neem de bus | omdat | het | hard | **regent** |
| Ik ga niet met de fiets | omdat | het | slecht weer | **is** |

### Сравнение:

| Want (глагол на месте 2) | Omdat (глагол в конце) |
|--------------------------|------------------------|
| Ik ga slapen want ik **ben** moe. | Ik ga slapen omdat ik moe **ben**. |
| Ik bel niet want ik **heb** geen tijd. | Ik bel niet omdat ik geen tijd **heb**. |

---

## 2. Als (Если/Когда)

**Als** работает так же, как **omdat** — глагол уходит **в конец**.

```
[Главное предложение] + als + [Подлежащее + Остальное + Глагол]
```

**Примеры:**
- Ze doet een stropdas aan **als** ze naar haar werk **gaat**.
  *(Она надевает галстук, когда идёт на работу.)*
- Ik neem de bus **als** het hard **regent**.
  *(Я еду на автобусе, если идёт сильный дождь.)*

## 3. Инверсия (Inversie)

### Что это?
Когда предложение начинается **НЕ с подлежащего**, глагол и подлежащее **меняются местами**.

### Обычный порядок:
```
Подлежащее + Глагол + Остальное
Ik          koop     een kaartje.
```

### Инверсия (начинаем с места/времени/вопроса):
```
[Место/Время/Вопрос] + Глагол + Подлежащее + Остальное
Op het station        koop     ik            een kaartje.
```

### Примеры:

| Начало предложения | Глагол | Подлежащее | Остальное |
|--------------------|--------|------------|-----------|
| **Op het station** | koop | ik | een kaartje |
| **Hoe laat** | kom | je | thuis? |
| **Met de trein** | ga | ik | naar het werk |
| **Elke dag** | gebruik | ik | de NS-app |

### ⚠️ Важно: Инверсия после придаточного предложения!

Когда предложение **начинается** с als/omdat/wanneer, в главном предложении тоже нужна инверсия!

```
[Придаточное (als/omdat...)], + [Глагол] + [Подлежащее] + [Остальное]
```

**Примеры:**

❌ **Неправильно:** Als ik vrij ben, **ik wandel** naar het park.
✅ **Правильно:** Als ik vrij ben, **wandel ik** naar het park.

❌ **Неправильно:** Omdat het regent, **ik neem** de bus.
✅ **Правильно:** Omdat het regent, **neem ik** de bus.

❌ **Неправильно:** Omdat mijn vrouw jarig is, **we kopen** een cadeautje.
✅ **Правильно:** Omdat mijn vrouw jarig is, **kopen we** een cadeautje.

## 4. Perfectum (Прошедшее время)

### Структура:
```
Подлежащее + hebben/zijn + ... + Participium (voltooid deelwoord)
```

### Когда HEBBEN, когда ZIJN?

| HEBBEN (большинство глаголов) | ZIJN (глаголы движения/изменения) |
|-------------------------------|-----------------------------------|
| Ik **heb** een boek **gelezen**. | Ik **ben** naar Amsterdam **gegaan**. |
| We **hebben** pizza **gegeten**. | Mila **is** naar Maastricht **gereisd**. |
| Ik **heb** de NS-app **gebruikt**. | Ze **zijn** met de trein **gekomen**. |

### Глаголы с ZIJN (запомнить!):

| Глагол | Perfectum | Перевод |
|--------|-----------|---------|
| gaan | zijn **gegaan** | идти, ехать |
| komen | zijn **gekomen** | приходить |
| rijden | zijn **gereden** | ехать (на машине/велосипеде) |
| vliegen | zijn **gevlogen** | летать |
| reizen | zijn **gereisd** | путешествовать |
| blijven | zijn **gebleven** | оставаться |
| worden | zijn **geworden** | становиться |
| zijn | zijn **geweest** | быть |

### ⚠️ Типичная ошибка:

❌ Mila en Simone **hebben** naar Maastricht **gegaan**.
✅ Mila en Simone **zijn** naar Maastricht **gegaan**.

## 5. Thuis vs Huis

| Слово | Тип | Использование | Пример |
|-------|-----|---------------|--------|
| **thuis** | наречие (bijwoord) | "дома" (где?) | Ik ben **thuis**. *(Я дома.)* |
| **huis** | существительное (zelfstandig naamwoord) | "дом" (что?) | Mijn **huis** is groot. *(Мой дом большой.)* |

### ⚠️ Типичная ошибка:

❌ Van mijn **thuis** naar de les...
✅ Van mijn **huis** naar de les...

**Правило:** После притяжательных местоимений (mijn, jouw, zijn...) используйте **huis**, не thuis!

## 6. Вопросительные слова с инверсией

| Вопросительное слово | Глагол | Подлежащее | Остальное |
|----------------------|--------|------------|-----------|
| **Hoe** | gaat | u | naar de les? |
| **Hoe laat** | komt | de trein | aan? |
| **Hoe lang** | bent | u | onderweg? |
| **Waarom** | blijf | je | thuis? |
| **Wat** | kost | een dagkaart | ? |

## 7. Scheidbare werkwoorden (Разделяемые глаголы)

Некоторые глаголы состоят из двух частей. В предложении они **разделяются**:

| Глагол | Пример |
|--------|--------|
| aan|doen | Ze **doet** een stropdas **aan**. |
| aan|komen | De trein **komt** om 10 uur **aan**. |
| mee|nemen | Ik **neem** mijn tas **mee**. |
| uit|stappen | We **stappen** in Amsterdam **uit**. |

### В придаточном предложении (с omdat, als, want) — слитно в конце:

- Ze is moe omdat ze vroeg **opstaat**. *(вместе в конце)*
- Ik neem de bus als ik mijn fiets niet **meeneem**.

---

## Краткая шпаргалка

| Правило | Запомни |
|---------|---------|
| **want** | Обычный порядок слов |
| **omdat, als** | Глагол → в конец |
| **Инверсия** | Не с подлежащего? → Глагол + Подлежащее |
| **После als/omdat в начале** | Главное предложение тоже с инверсией! |
| **Perfectum + движение** | ZIJN, не hebben |
| **thuis** | наречие ("дома") |
| **huis** | существительное ("дом") |
