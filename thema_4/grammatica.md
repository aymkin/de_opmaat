# Want en Omdat

# `Want` Structure

  [Hoofdzin] + want + [normale zin]

  The key thing about want: the word order after want stays normal (subject + verb).

  Normal Dutch word order:

  Subject + Verb + Rest

  | Subject | Verb   | Rest      |
  | ------- | ------ | --------- |
  | ik      | ben    | moe       |
  | het     | regent | hard      |
  | ik      | had    | geen tijd |

  Examples with "want":

  | Hoofdzin       | want | Subject | Verb   | Rest      |
  | -------------- | ---- | ------- | ------ | --------- |
  | Ik ga slapen   | want | ik      | ben    | moe       |
  | Ik neem de bus | want | het     | regent | hard      |
  | Ik bel niet    | want | ik      | heb    | geen tijd |

  Compare with "omdat" (different word order!):

  - Want: Ik ga slapen want ik ben moe. (verb stays in position 2)
  - Omdat: Ik ga slapen omdat ik moe ben. (verb goes to the END)
