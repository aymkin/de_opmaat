# Lijst met werkwoorden die het hulpwerkwoord ZIJN gebruiken

---

## 1. Beweging (van A naar B)

Deze gaan bijna altijd met **zijn**.

| Werkwoord   | Perfectum           | Перевод        |
| ----------- | ------------------- | -------------- |
| gaan        | is **gegaan**       | идти, ехать    |
| komen       | is **gekomen**      | приходить      |
| terugkomen  | is **teruggekomen** | возвращаться   |
| binnengaan  | is **binnengegaan** | входить        |
| uitgaan     | is **uitgegaan**    | выходить       |
| weggaan     | is **weggegaan**    | уходить        |
| opstaan     | is **opgestaan**    | вставать       |
| vertrekken  | is **vertrokken**   | отправляться   |
| aankomen    | is **aangekomen**   | прибывать      |
| blijven     | is **gebleven**     | оставаться     |
| verschijnen | is **verschenen**   | появляться     |
| emigreren   | is **geëmigreerd**  | эмигрировать   |
| arriveren   | is **gearriveerd**  | прибывать      |
| reizen      | is **gereisd**      | путешествовать |

---

## 2. Verandering van toestand

Deze beschrijven een situatie die **verandert**.

| Werkwoord                    | Perfectum           | Перевод       |
| ---------------------------- | ------------------- | ------------- |
| worden                       | is **geworden**     | становиться   |
| blijven (als toestand)       | is **gebleven**     | оставаться    |
| beginnen                     | is **begonnen**     | начинаться    |
| eindigen                     | is **geëindigd**    | заканчиваться |
| opgroeien                    | is **opgegroeid**   | вырастать     |
| sterven                      | is **gestorven**    | умирать       |
| overlijden                   | is **overleden**    | скончаться    |
| verdwijnen                   | is **verdwenen**    | исчезать      |
| ontstaan                     | is **ontstaan**     | возникать     |
| groeien                      | is **gegroeid**     | расти         |
| verbeteren (intransitief)    | is **verbeterd**    | улучшаться    |
| verslechteren (intransitief) | is **verslechterd** | ухудшаться    |

---

## 3. Onpersoonlijke / vaste werkwoorden

Deze krijgen altijd **zijn**, ook al veranderen ze niet van plaats.

| Werkwoord            | Perfectum                | Перевод             |
| -------------------- | ------------------------ | ------------------- |
| gebeuren             | is **gebeurd**           | происходить         |
| slagen (voor examen) | is **geslaagd**          | сдать (экзамен)     |
| zakken (voor examen) | is **gezakt**            | провалить (экзамен) |
| mislukken            | is **mislukt**           | не удаваться        |
| lukken               | is **gelukt**            | удаваться           |
| plaatsvinden         | **heeft** plaatsgevonden | происходить ⚠️      |
| vallen               | is **gevallen**          | падать              |
| opvallen             | is **opgevallen**        | бросаться в глаза   |

> ⚠️ **Uitzondering:** plaatsvinden → **heeft** plaatsgevonden (niet zijn!)

## ⭐ Extra: Werkwoorden met HEBBEN of ZIJN

Deze werkwoorden kunnen **beide** gebruiken:

- **hebben** = actie zonder richting
- **zijn** = beweging van A naar B

| Werkwoord | Met HEBBEN (actie)                               | Met ZIJN (richting)                                 |
| --------- | ------------------------------------------------ | --------------------------------------------------- |
| fietsen   | Ik **heb** gefietst. _(Ik was aan het fietsen.)_ | Ik **ben** naar school gefietst. _(van A naar B)_   |
| lopen     | Ik **heb** gelopen. _(beweging als sport)_       | Ik **ben** naar huis gelopen. _(van A naar B)_      |
| zwemmen   | Ik **heb** gezwommen. _(in het zwembad)_         | Ik **ben** naar de overkant gezwommen. _(richting)_ |
| rijden    | Ik **heb** gereden. _(autorijden als actie)_     | Ik **ben** naar Amsterdam gereden. _(van A naar B)_ |
| vliegen   | Het vliegtuig **heeft** gevlogen. _(actie)_      | Ik **ben** naar Parijs gevlogen. _(richting)_       |
| wandelen  | Ik **heb** gewandeld. _(als activiteit)_         | Ik **ben** naar het park gewandeld. _(richting)_    |

---

## ✔️ Korte regel

| Situatie                  | Hulpwerkwoord |
| ------------------------- | ------------- |
| Beweging (van A naar B)   | **ZIJN**      |
| Verandering van toestand  | **ZIJN**      |
| Actie zonder verplaatsing | **HEBBEN**    |

```
➡️ Gebruik ZIJN bij beweging of verandering.
➡️ Gebruik HEBBEN bij acties zonder verplaatsing.
```

## Voorbeelden uit Opdracht 2

| Zin                                                 | Waarom ZIJN?                 |
| --------------------------------------------------- | ---------------------------- |
| Mila en Simone **zijn** naar Maastricht **gegaan**. | gaan = beweging van A naar B |
| Ik **ben** met de fiets naar school **gereden**.    | rijden met richting          |
| De trein **is** om 10 uur **aangekomen**.           | aankomen = beweging          |
