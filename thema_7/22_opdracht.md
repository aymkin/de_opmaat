# 22 Grammatica en spreken

Werk in tweetallen. Beantwoord onderstaande vragen. In de vragen staat een
separabel verbum. Antwoord in complete zinnen, dus gebruik het separabel verbum
ook in je antwoord.

Voorbeeld

- Wat doe je aan als het regent? Dan doe ik een regenjas aan.

## A presens

1. Hoe laat sta je doordeweeks meestal op? → Ik **sta** doordeweeks meestal om
   zeven uur **op**.

2. Slaap je in het weekend vaak uit? → Ja, in het weekend **slaap** ik vaak
   **uit** tot tien uur.

3. Ga je komend weekend met vrienden uit? → Ja, ik **ga** komend weekend met
   vrienden **uit**.

4. Als je jarig bent, nodig je dan vrienden en familie voor je verjaardag uit? →
   Ja, als ik jarig ben, **nodig** ik vrienden en familie voor mijn verjaardag
   **uit**.

5. Moet je weleens een Nederlands formulier invullen? Waarvoor? → Ja, ik moet
   weleens een formulier **invullen**, bijvoorbeeld voor de gemeente.

6. Doe je je schoenen uit als je thuis bent of hou je ze aan? → Ik **doe** mijn
   schoenen **uit** als ik thuis ben.

7. Wat neem je altijd mee naar de cursus? → Ik **neem** altijd een schrift en
   een pen **mee** naar de cursus.

8. Lever je je huiswerk digitaal of op papier in? → Ik **lever** mijn huiswerk
   op papier **in**.

9. Met wie werk je in de cursus vaak samen? → Ik **werk** in de cursus vaak met
   mijn buurman **samen**.

10. Wie maakt meestal bij jou thuis het eten klaar? En vanavond ook? → Mijn
    partner **maakt** meestal het eten **klaar**. Ja, vanavond ook.

## B perfectum

1. Wanneer ben je voor het eerst in Nederland aangekomen? → Ik ben in 2024 voor
   het eerst in Nederland **aangekomen**.

2. Hoelang geleden heb je je middelbare school afgerond? → Ik heb mijn
   middelbare school vijftien jaar geleden **afgerond**.

3. Ben je afgelopen weekend uitgegaan? Zo ja, waar naartoe? → Ja, ik ben
   afgelopen weekend **uitgegaan** naar een restaurant.

4. Heb je afgelopen weekend uitgeslapen? → Ja, ik heb afgelopen weekend
   **uitgeslapen** tot elf uur.

5. Hoe laat ben je vanochtend opgestaan? → Ik ben vanochtend om half acht
   **opgestaan**.

6. Hoe laat ben je vanochtend de deur uitgegaan? → Ik ben vanochtend om kwart
   over acht de deur **uitgegaan**.

7. Hebben Nederlandse vrienden je wel eens voor een feest uitgenodigd? → Ja,
   Nederlandse vrienden hebben mij een keer voor een feest **uitgenodigd**.

8. Wanneer heb je je voor deze cursus ingeschreven? → Ik heb me in januari voor
   deze cursus **ingeschreven**.

9. Wat heb je naar de cursus meegenomen? → Ik heb mijn boek en een woordenboek
   naar de cursus **meegenomen**.

10. Heb je het huiswerk voor deze les afgemaakt? → Ja, ik heb het huiswerk voor
    deze les **afgemaakt**.
