# Grammaticale Analyse — Daily Standup 18 december 2025

## Overzicht van tijden in de tekst

| Tijd                             | Frequentie | Gebruik                     |
| -------------------------------- | ---------- | --------------------------- |
| **Tegenwoordige tijd (OTT)**     | ~70%       | Huidige situatie, gewoontes |
| **Voltooide tijd (VTT)**         | ~15%       | Afgeronde acties            |
| **Toekomst (gaan + infinitief)** | ~10%       | Plannen, intenties          |
| **Verleden tijd (OVT)**          | ~5%        | Verhalen over gisteren      |

---

## 1. TEGENWOORDIGE TIJD (Onvoltooid Tegenwoordige Tijd)

### Meest gebruikte werkwoorden

#### ZIJN (to be) — onregelmatig

| Persoon | Vorm | Voorbeeld uit tekst                           |
| ------- | ---- | --------------------------------------------- |
| ik      | ben  | "Ik **ben** daar mee bezig"                   |
| jij/je  | bent | "**Ben** je mee bezig?"                       |
| hij/zij | is   | "Mike **is** er wel" / "Het **is** donderdag" |
| wij     | zijn | "We **zijn** net begonnen"                    |
| jullie  | zijn | —                                             |
| zij     | zijn | "Er **zijn** geen meetings"                   |

**Constructie: zijn + bezig (met)**

```
Ik ben bezig           — Я занят
Ik ben ermee bezig     — Я этим занимаюсь
Ik ben daar mee bezig  — Я этим занимаюсь (emphasis)
```

---

#### HEBBEN (to have) — onregelmatig

| Persoon | Vorm   | Voorbeeld uit tekst                    |
| ------- | ------ | -------------------------------------- |
| ik      | heb    | "Ik **heb** hem gewijzigd"             |
| jij/je  | hebt   | "**Heb** je nog iets nodig?"           |
| hij/zij | heeft  | "Hij **heeft** comments achtergelaten" |
| wij     | hebben | "We **hebben** de daily"               |

---

#### GAAN (to go) — voor toekomst

| Persoon | Vorm | Voorbeeld uit tekst                    |
| ------- | ---- | -------------------------------------- |
| ik      | ga   | "Ik **ga** vandaag reviewen"           |
| jij/je  | gaat | "Jij **gaat** dingen afronden"         |
| hij/zij | gaat | "Dit **gaat** los van de app"          |
| wij     | gaan | "We **gaan** naar een bord met labels" |

**Structuur: gaan + infinitief = toekomst**

```
Ik ga reviewen         — Я буду делать review
Ga ik mee verder       — Я продолжу (inversie!)
We gaan features maken — Мы будем делать features
```

---

#### KUNNEN (can/to be able) — modaal

| Persoon | Vorm     | Voorbeeld uit tekst              |
| ------- | -------- | -------------------------------- |
| ik      | kan      | "Ik **kan** het vandaag afmaken" |
| jij/je  | kan/kunt | "Wat je **kan** doen..."         |
| hij/zij | kan      | "Het **kan** zijn dat Rafal..."  |
| wij     | kunnen   | "We **kunnen** features maken"   |

---

#### MOETEN (must/have to) — modaal

| Persoon | Vorm | Voorbeeld uit tekst                 |
| ------- | ---- | ----------------------------------- |
| ik      | moet | "Ik **moet** vandaag verder zoeken" |
| het     | moet | "Het **moet** nog getest worden"    |

---

### Regelmatige werkwoorden (stam + t/en)

| Infinitief | Stam  | Hij/zij | Wij     | Voorbeeld                               |
| ---------- | ----- | ------- | ------- | --------------------------------------- |
| werken     | werk  | werkt   | werken  | —                                       |
| wachten    | wacht | wacht   | wachten | "even wachten op feedback"              |
| staan      | sta   | staat   | staan   | "de link **staat** na wijzigen"         |
| komen      | kom   | komt    | komen   | "Die ticket **komt** hierna"            |
| liggen     | lig   | ligt    | liggen  | "De release **ligt** bij Suzanne"       |
| denken     | denk  | denkt   | denken  | "Ik **denk** dat het niet veel werk is" |
| duren      | duur  | duurt   | duren   | "misschien **duurt** het een paar uur"  |

## 2. VOLTOOIDE TIJD (Voltooid Tegenwoordige Tijd)

### Structuur: hebben/zijn + voltooid deelwoord

```
HEBBEN + ge...d/t    (de meeste werkwoorden)
HEBBEN + ge...en     (sterke werkwoorden)
ZIJN + ge...d/t      (beweging/verandering)
```

### Voorbeelden uit de tekst

| Zin                                         | Analyse                                      |
| ------------------------------------------- | -------------------------------------------- |
| "Ik **heb** hem **gewijzigd**"              | hebben + ge-wijzig-d (zwak)                  |
| "Hij **heeft** comments **achtergelaten**"  | hebben + achter-ge-laten (scheidbaar, sterk) |
| "Ik **heb** alles **overgedragen**"         | hebben + over-ge-dragen (scheidbaar, sterk)  |
| "Daar **is** feedback op **geweest**"       | zijn + geweest (zijn = beweging/staat)       |
| "We **zijn** net **begonnen**"              | zijn + begonnen (sterk)                      |
| "**Is** de vergadering al **begonnen**?"    | zijn + begonnen                              |
| "Kennet **heeft** zijn Epic **aangemaakt**" | hebben + aan-ge-maakt (scheidbaar)           |

### Voltooid deelwoord — Patronen

#### Zwakke werkwoorden: ge + stam + d/t

```
wijzigen  → ge-wijzig-d
testen    → ge-test (stam eindigt op -t)
maken     → ge-maakt
```

#### Sterke werkwoorden: ge + stam + en (met klinkerverandering)

```
beginnen  → begonnen (i → o)
komen     → gekomen
geven     → gegeven
laten     → gelaten
dragen    → gedragen
```

#### Scheidbare werkwoorden: prefix + ge + stam

```
aan|maken     → aan-ge-maakt
achter|laten  → achter-ge-laten
over|dragen   → over-ge-dragen
op|pakken     → op-ge-pakt
af|ronden     → af-ge-rond
```

#### Zonder ge- (niet-scheidbare prefixen: be-, ver-, ont-, er-, ge-, her-)

```
beginnen   → begonnen (geen ge-)
overleggen → overlegd (geen ge-)
```

## 3. INVERSIE (Omkering)

### Regel: Als de zin NIET met subject begint → werkwoord vóór subject

| Normaal                | Met inversie                   | Reden         |
| ---------------------- | ------------------------------ | ------------- |
| Ik ga vandaag reviewen | **Vandaag** ga **ik** reviewen | Tijd voorop   |
| Ik ben daar mee bezig  | **Daar** ben **ik** mee bezig  | Plaats voorop |
| Het komt hierna        | **Hierna** komt **het**        | Tijd voorop   |
| Ik ga mee verder       | **Ga** ik mee verder           | (korte vorm)  |

### Voorbeelden uit de tekst

```
"Gisteren heeft iemand de review gedaan"
        ↓
    [Tijd] + [werkwoord] + [subject] + [rest]

"Na vandaag ben ik 2,5 week weg"
     ↓
  [Tijd] + [werkwoord] + [subject] + [rest]

"Voor mij geldt hetzelfde"
    ↓
 [Voorwerp] + [werkwoord] + [subject]
```

## 4. SCHEIDBARE WERKWOORDEN

### In de tekst gevonden:

| Werkwoord        | Scheidbaar | Voorbeeld                            |
| ---------------- | ---------- | ------------------------------------ | --------------------------------------- |
| **beginnen met** | nee        | "Ben ik vanochtend aan **begonnen**" |
| \*\*aan          | maken\*\*  | ja                                   | "Kennet heeft zijn Epic **aangemaakt**" |
| \*\*achter       | laten\*\*  | ja                                   | "Hij heeft comments **achtergelaten**"  |
| \*\*over         | dragen\*\* | ja                                   | "Ik heb alles **overgedragen**"         |
| \*\*af           | ronden\*\* | ja                                   | "Dingen **afronden** voor vakantie"     |
| \*\*op           | pakken\*\* | ja                                   | "Vest die aan het **oppakken** is"      |
| \*\*door         | gaan\*\*   | ja                                   | "We gaan **door** naar..."              |
| \*\*mee          | kijken\*\* | ja                                   | "kunnen wij hier naar **kijken**"       |
| \*\*na           | denken\*\* | ja                                   | "Had ik nog niet over **nagedacht**"    |

### Regels voor scheidbare werkwoorden

**1. Hoofdzin (OTT):** prefix gaat naar het einde

```
Ik maak een ticket aan.
   ↓           ↓
 [stam]    [prefix]
```

**2. Voltooide tijd:** ge- komt tussen prefix en stam

```
aan + ge + maakt = aangemaakt
op + ge + pakt = opgepakt
```

**3. Met modaal werkwoord:** blijft samen

```
Ik kan het vandaag afmaken.
                   ↓
              [samen aan einde]
```

## 5. ER-CONSTRUCTIES

### Voorbeelden uit de tekst:

| Zin                              | Analyse                          |
| -------------------------------- | -------------------------------- |
| "Er hangt nog geen feature aan"  | er = plaatshouder (existentieel) |
| "Er is niet veel voortgang"      | er + zijn = "there is"           |
| "Er zijn geen meetings"          | er + zijn = "there are"          |
| "Mike is **er** wel"             | er = daar (locatie)              |
| "Ik ben **ermee** bezig"         | er + mee = daarmee               |
| "Had ik nog niet over nagedacht" | (erover weggelaten)              |

### Er-combinaties

```
ermee   = er + met   = daarmee (with it)
erover  = er + over  = daarover (about it)
erin    = er + in    = daarin (in it)
erop    = er + op    = daarop (on it)
eraan   = er + aan   = daaraan (to it)
```

## 6. VRAAGZINNEN

### Ja/nee-vragen: werkwoord voorop

```
Ben je mee bezig?           — Ты этим занимаешься?
Heb je nog iets nodig?      — Тебе ещё что-то нужно?
Moet de Deploy Preview...?  — Нужно ли Deploy Preview...?
```

### W-vragen: vraagwoord + werkwoord + subject

```
Wat heb je?                 — Что у тебя? (Что скажешь?)
Hoe test ik dat dan?        — Как мне тогда это тестировать?
Wat je kan doen...          — Что ты можешь сделать...
```

## 7. HANDIGE PATRONEN UIT DE DAILY

### Status rapporteren

```
Ik ben ermee bezig              — Я этим занимаюсь
Ik ben daar mee bezig           — Я над этим работаю
Ben ik vanochtend aan begonnen  — Я начал этим утром
Ga ik mee verder                — Продолжу этим заниматься
Het komt hierna                 — Это будет следующим
```

### Voltooide acties

```
Ik heb hem gewijzigd            — Я его изменил
Hij heeft de review gedaan      — Он сделал review
Ik heb alles overgedragen       — Я всё передал
```

### Plannen

```
Ik kan het vandaag afmaken      — Я могу закончить сегодня
Ik ga vandaag reviewen          — Я сегодня буду делать review
Misschien duurt het een paar uur — Может займёт пару часов
```

### Problemen beschrijven

```
Er is niet veel voortgang       — Нет большого прогресса
Dat lukt niet                   — Это не получается
Het is lastiger te testen       — Это сложнее тестировать
```

### Strategieën

```
Ik heb een andere strategie gekozen — Я выбрал другую стратегию
Nu doe ik het in kleine stapjes     — Теперь делаю маленькими шагами
Stap voor stap                      — Шаг за шагом
```

## Samenvatting — Wat te onthouden

1. **OTT is de basis** — de meeste zinnen in een daily zijn in de tegenwoordige
   tijd
2. **VTT voor updates** — "Ik heb X gedaan" voor wat je al hebt afgerond
3. **Gaan + infinitief** — voor plannen: "Ik ga X doen"
4. **Inversie** — als je met tijd/plaats begint: "Vandaag ga ik..."
5. **Scheidbare werkwoorden** — let op de positie van het prefix!
6. **Er-constructies** — heel typisch Nederlands, veel gebruiken!
